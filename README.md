# Shirt_shop_ecommerce
 Ecommerce where users can build their store and sell their customized shirts.
 
* Front End : HTML5, CSS3, Bootstrap,JQueqy, Angular2.
* Back End : NodeJS, MongoDB. 


# Run Project
* Install node
* Install npm
* Install Angular-Cli
* Clone Project
* In folder project run : npm install
* run ng serve
* localhost:2400
